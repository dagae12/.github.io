📝 使用方法
================================================================

【安装部署】

本地测试：
1. 解压 zhugehui_v5.0_alpha.zip
2. 用浏览器打开 zhugehui_v5.0_alpha.html
3. 开始测试

在线部署：
1. 访问 app.netlify.com/drop
2. 拖入解压后的文件夹
3. 获得永久URL
4. 添加到手机主屏幕

【功能使用】

段位查看：
→ 状态标签 → 段位系统卡片 → 查看当前段位

活跃度打卡：
→ 每日标签 → 勾选生活基础任务 → 看分数变化

保存记录：
→ 每日标签 → 填写信息 → 保存记录 → 获得经验+宝箱

购物记录：
→ 财务标签 → 填写详细信息 → 选择类别/平台/必要性 → 保存
