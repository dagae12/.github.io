🎯 重点测试项
================================================================

✅ 看段位卡片
   → 进入APP
   → 点击"状态"标签
   → 查看"段位系统"卡片
   → 看到你当前的段位（比如：黑铁III）

✅ 试活跃度
   → 点击"每日"标签
   → 勾选"早上洗脸刷牙"
   → 勾选"吃早餐/午餐/晚餐"
   → 勾选"晚上洗澡"
   → 看活跃度分数实时变化

✅ 保存记录获得宝箱
   → 在"每日"页面
   → 填写起床时间
   → 选择学习时长1小时
   → 选择娱乐放松30分钟
   → 点击"保存记录"
   → 获得宝箱奖励！

✅ 查看段位进步
   → 回到"状态"标签
   → 看经验条增加
   → 看段位进度变化
   → 升级时段位自动晋升

✅ 测试购物记录
   → 点击"财务"标签
   → 填写物品名称（比如：机械键盘）
   → 填写金额
   → 选择支出
   → 选择类别（比如：电子）
   → 选择平台（比如：京东）
   → 选择必要性（比如：想要的）
   → 保存

💡 新功能亮点
================================================================

1. 30个小段位
   - 每升1级都有段位变化
   - 🔴黑铁 → 🟤黄铜 → ⚪白银 → 🟡黄金
   - 💎铂金 → 💠钻石 → ⭐大师 → 🔥宗师 → 👑王者

2. 每日活跃度
   - 洗脸刷牙、三餐、洗澡都有分数
   - 学习工作、娱乐放松都计入
   - 达到60/80/95分自动获得宝箱

3. 宝箱奖励
   - 普通宝箱：看电影、吃大餐
   - 精良宝箱：KTV、剧本杀
   - 稀有宝箱：短途旅行、SPA
   - 更高级宝箱有更棒奖励！

4. 连续登录
   - 每天打开APP自动检测
   - 连续3天：普通宝箱
   - 连续7天：精良宝箱+称号
   - 连续30天：史诗宝箱！

5. 购物记录2.0
   - 记录买了什么
   - 分析消费习惯
   - 知道钱花在哪里

❓ 常见问题
================================================================

Q: 我的数据会丢失吗？
A: 不会！v5.0完全兼容v4.0数据，自动迁移。

Q: 如何看我的宝箱？
A: 目前宝箱自动发放到库存，v5.0 Beta会有查看界面。

Q: 段位怎么升？
A: 正常升级就会自动晋升段位，每升1级段位+1。

Q: 活跃度每天重置吗？
A: 是的，每天0点自动重置，重新开始计分。

Q: 登录奖励什么时候发？
A: 每次打开APP时自动检测并发放。

Q: BUG怎么反馈？
A: 记录BUG详情，告诉开发者即可。
