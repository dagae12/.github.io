🔧 技术实现
================================================================

【HTML】
```html
<img id="rankIcon" src="rank_iron.png" 
     style="width:100%; height:100%; object-fit:contain;">
```

【JavaScript】
```javascript
const RANKS=[
    {name:'黑铁III', img:'rank_iron.png', ...},
    {name:'黄铜IV', img:'rank_bronze.png', ...},
    ...
];

// 在render函数中
rankIcon.src = currentRank.img;
```

【自动切换】
- 等级变化 → updateRank() → render()
- render()读取当前段位的img字段
- 自动更新img标签的src
- 图片平滑切换

================================================================

【方法1：本地测试】
1. 解压 zhugehui_v5.0_fixed.zip
2. 确保所有文件在同一目录
   - index.html
   - rank_*.png (9个徽章)

4. 查看段位徽章

【方法2：在线部署】
1. 访问 app.netlify.com/drop
2. 解压后将整个文件夹拖入
3. 重要：确保所有图片文件都在同一目录

5. 添加到手机主屏幕
